# Chat application
Node.js application using socket.io

## Requirements

- node.js 12.*
- npm 6.*

## Dependencies

- express 4.17.1
- socket.io 2.3.0

## Building

Install the dependencies before start:

```sh
$ cd node-socketio
$ sudo npm install
```

## Client javascript libraries

- mustache 3.0.1
- moment 2.22.2
- qs 6.6.0

## Run the Application

```sh
$ npm start
```
